Makefile

	Makefile to build the UPC required library specification.

upc-lib-collectives.tex

	UPC collectives specification.  UPC collectives are a component of
	the UPC required library.

upc-lib-required-spec.tex

	The UPC required library specification driver file.

upc-lib-required-draft-spec.tex

	The UPC required library specification draft version driver file.
	This driver sets the draft flag and then includes the main driver.

upc-lib-tick.tex

	High-Performance Wall-Clock Timers specification.  Walk clock
	timers were introduced in UPC specification version 1.3.
